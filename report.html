Introduction

DigiPass is an Android based Digital Gate Pass Management System developed by Nexyes.in. It replaces the traditional paper-based gate pass process by providing a secure QR Code approval workflow for students, Teaching Guardians, HODs, Administrators and Security Guards.

Objectives

  • Paperless Gate Pass
  • QR Code Verification
  • Multi College Support
  • Secure Approval Workflow
  • Cloud Based Architecture

System Workflow

  1. Student submits request
  2. TG Approval
  3. HOD Approval
  4. QR Generated
  5. Security Verification
  6. Gate Pass Expires after 8 Hours
